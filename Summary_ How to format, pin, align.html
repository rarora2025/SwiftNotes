FULL SUMMARY- Formatting/Pinning/Alligning the Screen

SAFE AREA/SUPERVIEW: (Pinning/Constraining Single objects)

Go to constraints (square w/ lines around it)- set constraints to 0 on all sides

To make it cover the whole screen on landscape, select constraints >> trailing/leading- and make them equal to SUPERVIEW, also make sure the constant is set to 0 .

ALLIGNING:

Go to alignment (bar graphs) and click horizontally/vertically in container

CONTAINERS/SUBVIEWS/STACK VIEWS (embedding views)- FULL STEPS

  1. Drag new view/embed objects in new subview based on location

  2. Hold on all views and embed them in a new stack view

  3. Give stack view constraints- go to square w/ lines around it and set constraints to 0 on all sides- To make it cover the whole screen on landscape, select constraints >> trailing/leading/bottom/top- and make them equal to SUPERVIEW, also make sure the constant is set to 0 .

  4. Go to stack view, attributes, and select distribution to fill equally

  5. Now make sure that all objects in the same stack view have their own stack views, and then select those stack views/those solo objects in their own stack view, and make those vertically/horizontally aligned in their container, or pinned

  6. Select all containers, change background to default/transparent

  7. DONE!!

HOW TO CLEAN UP YOUR BUTTONS IF THEY ARE TIGHTENED UP AFTER PUT IN STACK VIEWS:

  1. Select the button

  2. Go to constraints (square w/ lines on side), and change the width to custom sizes chosen; Add those constraints

  3. Look at warning (fixed width constraint)- click on error and set the constraint to being greater than custom width so it will automatically adjust
