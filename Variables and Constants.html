Working with Variables

Defining Variables:

var typedVariable: Type
var myVariable = 42

Defining Constants:

let myConstant = 12
let typedConstant: Type

How to insert a variable into text:

Use \ backslash then parenthesis w/ variable

var age:int=1

var pet:String="dog"

print("Your \(pet) is \(age) years old!”)

If you aren’t using a variable, you can name It: _
